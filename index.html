Strings.ai Assignment


Purpose: To create a page layout similar to Netflix, but filled with Gifs of various categories using Giphi API

Implementation Requirements: There should be a list of categories and the images under each category are carousel. The categories are lazy loaded, as and when the user scrolls the page.

Instructions to use: Click on the "Click me" button to get started. This button will be disabled once it is clicked. Please observe the console to observe the lazy loading as the page is being scrolled. You can alter the lazy loading feature by changing the scrollAmount variable in the javascript file.
